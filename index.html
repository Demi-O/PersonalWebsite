Undergraduate Research Assistant

Co-op Job

May - August

Developed an application that was used for research into data exploration techniques of geotagged information on a tabletop device. I worked with another coop student to complete this, the application involves two way communication between a tabletop and a number of tablets.
  • Faciliated the addition of items to the startup screen and map of the table by reading information from csv files.
  • Added functions to the server and the webpages behind the table and tablet applications to send information across the devices.
  • Improved the indirect manipulation of the bounding box on the table by making the tablet function as a joystick.
  • Added flashing animations and icon outlining to draw the user's attention to certain icons on the table when specific actions were performed on the tablet.
  • Added switching between the two modes of the tablet.
  • Used Processing to decipher log data files from the main application and create plain text and Micrososft Excel files filled with useful information.
  • Used Processing to create visualizations from the log files that were needed for the paper, these visualizations were image time heatmaps and manipulation type heatmaps.
